# public-employee-salary
# 📊 Employee Salary Dashboard (Power BI)

## 📝 Overview
This project is an interactive Power BI dashboard built using an employee salary dataset.  
It provides insights into employee count, salary distribution, and job-level analysis.

---

## 📂 Dataset
The dataset includes employee details such as name, job role, salary, grade, and employment type.

### Columns Used:
- First Name
- Last Name
- Job Title
- Department
- Grade
- Annual Salary
- Hire Date
- Employment Type
- Employee ID

---

## 📊 Dashboard Features

### 🔹 Key Metrics
- Total Employees: **41K**
- Total Salary: **3.62 Billion**
- Employee Count calculated using **COUNT(Employee ID)**

---

### 🔹 Visualizations

#### 1. Salary by Job Title
- Bar chart showing total salary distribution across job roles

#### 2. Salary by Grade
- Column chart showing how salary varies by grade level

#### 3. Employee Count
- Displays total number of employees in the organization

---

## ⚙️ Data Processing
- Renamed columns for better understanding
- Converted date column into proper format
- Used:
  - COUNT(Employee ID) for total employees
  - SUM(Annual Salary) for total salary

---

## 📈 Key Insights
- Higher job roles have higher salary distribution
- Salary increases with grade level
- Organization has approximately 41,000 employees
